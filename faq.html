Clients

Création de compte

Créer un compte client

Créer un compte Trendigo se fait à partir de l’Application mobile qui peut être téléchargée sur un téléphone intelligent Android ou Iphone. Quelques informations sont nécessaires. Vous recevez ensuite un courriel avec un code d’authentification que vous devez entrer dans votre application mobile. C’est aussi simple. Pour pouvoir utiliser son compte, c’est à dire payer et recevoir du cash-back sur ses porte-monnaies électroniques, il est nécessaire d’associer une carte bancaire. Les informations sont cryptées et rendues non accessibles.

Mon compte a été refusé

Un compte n’est jamais refusé. Par contre, il est possible qu’un compte ne puisse pas être créé pour l’une des raisons suivantes :
- L’adresse n’existe pas. Ceci arrive quand l’adresse n’est pas entrée correctement. Vérifiez notamment l’orthographe et le code postal.
- Le courriel n’est pas valable. La validation du compte se fait à partir d’un courriel qui vous est envoyé pour confirmer votre identité. Si l’adresse est fausse ou mal orthographiée, vous ne pouvez pas le recevoir et donc pas la valider.
- Il n’y a pas de moyen de paiement associé à votre compte. Pour être effectif, un compte Trendigo doit être couplé à une carte de paiement. Cela n’empêche pas la création du compte, mais son utilisation. Pour y remédier, ajouter une carte et vérifiez, si vous en avez une, qu’elle est bien valide.

Puis-je créer plusieurs comptes ?

Il n’est pas possible de créer plusieurs comptes avec le même courriel. Il est possible de créer plusieurs comptes, mais ceci n’est pas nécessaire. Il n’est pas nécessaire de créer plusieurs comptes Trendigo, même si vous allez chez des marchands différents. L’application gère les paiements, les récompenses sous forme de cash-back et les attribue aux marchands chez qui vous pouvez les utiliser. Vous n’avez rien à faire en dehors de scanner le code QR au moment de payer. Si vous souhaitez utiliser plusieurs cartes, nul besoin de créer plusieurs comptes. Vous pouvez ajouter autant de cartes que vous souhaitez et choisir celle avec laquelle vous allez régler, y compris à chaque paiement.

Gestion du compte

Modifier des données dans mon compte

La modification des informations de mon compte se fait à partir de l’application mobile. Je peux mettre à jour mes coordonnées. Il suffit de cliquer sur le profil et de sélectionner l’information à modifier. Vous pouvez aussi supprimer le compte à partir de cette zone.

Changer la photo de mon compte

Pour changer la photo du compte, il suffit de cliquer dessus et de prendre une nouvelle photo ou de la télécharger depuis le téléphone.

Accéder à mes transactions

Pour voir l’ensemble des paiements réalisés avec Trendigo, il suffit de cliquer sur le menu « transactions ». La liste des transactions est disponible par date de réalisation.

Pourquoi ai-je plusieurs porte-monnaies électroniques (e-wallets) ?

Tous les clients Trendigo ont un porte-monnaie principal sur lequel sont chargés les montants reçus en cash-back des marchands membres de la communauté Trendigo. Comme l’application de paiement Trendigo est un moyen de paiement universel, il sert aussi à payer chez des marchands, des groupements de clients ou des franchises acceptant ce moyen de paiement. Chacun d’eux a son propre porte-monnaie qui cumule les montants qui peuvent être dépensés chez eux. Vous n’avez pas à gérer ces porte- monnaies. Chaque fois que vous souhaitez payer, l’application vous informe du montant disponible chez ce commerçant et vous demande si vous voulez l’utiliser.

Paiement

Comment ça marche ?

L’application de paiement Trendigo est un moyen de paiement qui permet de régler ses factures chez les commerçants qui l’acceptent. Pour l’utiliser, un client doit télécharger l’application sur son téléphone intelligent, créer un compte et ajouter une carte de paiement.
Pour régler, il faut appuyer sur le bouton « Payer » situé en bas des pages de l’application et de scanner le Code QR Trendigo chez le commerçant. Deux code QR existent :
- Le code unique propre au commerçant.
Scanner le code.
Entrer le montant à payer dans la page qui s’ouvre, puis, s’il y a lieu, le pourboire que vous voulez laisser, choisir la carte de paiement (par défaut, la dernière utilisée), utiliser ou non les points Trendigo
Confirmer votre paiement.
- Le code créé par le commerçant sur son téléphone.
Scanner le code sur le téléphone du commerçant, vérifier que le montant est le bon, puis, s’il y a lieu, le pourboire que vous voulez laisser et appuyer sur payer.
A chaque paiement, vous pouvez choisir d’utiliser le cash-back disponible dans votre porte-monnaie.
Le solde entre le montant à payer et la somme présente dans le porte-monnaie est débité sur votre carte.

Où trouver le code QR chez le commerçant ?

Le code de paiement est en général bien placé par le commerçant dans un endroit visible et facile d’accès. Le code QR peut aussi être généré par le commerçant. Dans ce cas, il présentera son téléphone et vous devrez scanner son code QR.

Le commerçant me demande de scanner un code sur son mobile

Oui, cela est tout à fait normal. Le commerçant a deux possibilités. Soit il utilise son code unique, soit il génère un code à chaque paiement, Dans ce second cas, c’est lui qui entre le montant à payer et vous présente son téléphone.

Ajouter, supprimer ou mettre à jour une carte de paiement

Les cartes de paiement peuvent être ajoutées ou supprimées à partir de l’application mobile. Il suffit de suivre les instructions dans le menu cartes. Pour la supprimer, glissez la carte à gauche avec votre doigt et appuyez sur la poubelle. Vous pouvez ajouter autant de cartes que vous voulez, mais il faudra toujours en avoir une qui servira par défaut.

Puis-je ajouter une carte de débit ?

Oui, les cartes de débit Mastercard et Visa peuvent être ajoutées pour servir de paiement avec l’application Trendigo. Les cartes bancaires Interac le seront prochainement.

Quelles sont les cartes acceptées ?

L’application Trendigo accepte toutes les cartes de crédit Visa, Mastercard, American Express et les cartes de débit Visa ou Mastercard. D’autres cartes pourraient être acceptées dans le futur.

Où est transféré l’argent reçu en cash-back ?

Chaque fois que vous effectuez un paiement chez un commerçant qui participe au programme de récompense, le montant du cash-back qu’il a choisi de vous remettre est placé dans le porte-monnaie électronique correspondant. Il est immédiatement disponible pour un autre paiement. Si c’est un marchand indépendant, un membre d’un groupe de marchands ou d’une franchise, le montant sera crédité dans le porte-monnaie du marchand, du groupe ou de la franchise et pourra être utilisé immédiatement pour un autre achat chez eux. En tout temps, il est possible de voir les montants crédités dans le menu e-wallets.

Puis-je utiliser l’argent reçu en cash-back partout ?

Le montant de cash-back peut être utilisé chez le marchand qui vous a remis le montant ou dans le réseau auquel il appartient. Ce montant peut être utilisé immédiatement pour un autre achat chez eux.

Quand puis-je décider d’utiliser l’argent reçu en cash-back ?

L’argent reçu en cash-back est immédiatement disponible pour un autre paiement. Vous pouvez choisir de l’utiliser en cliquant sur le bouton de sélection au moment de l’achat, après la sélection de la carte.

Réseau de marchands

Quels sont les marchands qui acceptent Trendigo ?

Tous les marchands qui acceptent les paiements Trendigo sont indiqués et géolocalisés dans l’application. Il s’agit de marchands très divers, souvent de proximité, soucieux de leur quartier et offrant des services de qualité à leurs clients.

Quelle différence entre la communauté Trendigo et les réseaux privés de commerçants ?

Le nombre de marchands qui acceptent Trendigo est de plus en plus grand. La grande majorité offre un montant de cash-back sur les achats réalisés dans leur commerce. Certains commerces, réseaux de commerçants ou franchises ont créé leur propre programme de fidélisation en limitant l’usage du cash-back qu’ils versent aux paiements fait dans leur propre réseau privé. Pour un client, tout est transparent et c’est l’application qui gère l‘attribution et l’usage des montants disponibles selon le réseau du marchand.

Marchands

Création de compte

Créer un compte marchand

Un compte doit être créé en ligne sur le site sécurisé Trendigo. Ouvrir un compte marchand Trendigo est à la fois très sécuritaire et très simple. Il suffit de rentrer ses informations d’entreprise en suivant les indications du formulaire puis les informations bancaires. Toutes ces données sont cryptées et remplacées par des jetons qui servent à valider les transactions. Très rapidement, nous vérifions l’identité de votre entreprise et sa conformité de manière à accepter immédiatement votre inscription.

Documents officiels pour création d’un compte marchand

Afin de certifier votre entreprise, nous vous demandons quelques informations ainsi qu’un specimen de chèque.

Mon compte a été refusé

Notre priorité est la sécurité et la confidentialité des paiements tant pour les clients que pour les commerçants. Si votre compte n’a pas été validé, il peut manquer une information importante. Vérifiez à nouveau les informations fournies. Si vous n’êtes toujours pas en mesure de créer un compte, communiquez avec notre service client au 514 766 5005 ou par courriel à info@trendigo.com.

Entrer mes coordonnées bancaires

Il est demandé d’entrer les coordonnées bancaires au moment de la création de votre compte. Les numéros à entrer correspondent aux numéros suivants : Numéro de banque | Numéro de transit | Numéro de compte. Vous pouvez toujours modifier le compte après votre inscription.

Puis-je créer plusieurs comptes ?

Non, un commerçant n’a pas besoin de créer plusieurs comptes. Cependant, nous répondons à toute demande d’adaptation à des situations spécifiques. Communiquez avec nous et notre équipe d’implantation vous assistera.

Quel forfait choisir ?

Vous pouvez uniquement accepter les paiements trendigo et rejoindre la communauté. Si vous souhaitez créer votre propre système de fidélisation et restreindre l’usage du cashback à votre commerce, choisissez « créer votre programme ». Vous pourrez y inviter d’autres commerçants. Enfin, si on vous a offert de rejoindre un programme existant, choisissez « rejoindre un programme » et entrez son nom.

Gestion du compte

Accéder au tableau de bord marchand

Une fois votre compte créé, vous pouvez accéder au tableau de bord en entrant votre login et mot de passe. Tout peut y être géré.

Modifier des données dans mon compte

La plupart des informations de votre compte peuvent être modifiées depuis le back- office. Une validation vous sera demandée après chaque changement de données sensibles comme votre compte bancaire ou votre activité).

Modifier mes coordonnées bancaires

Vous pouvez modifier les coordonnées du compte bancaire lié à votre compte dans la rubrique Informations bancaires . Une validation vous sera demandée.

Changer la photo de mon compte

Vous pouvez changer la photo de profil du compte en haut à droite, en cliquant sur la photo. Vous pouvez changer les photos de présentation de votre compte dans la rubrique Mon Profil | Informations sur le commerce.

Gérer les taux de cash-back que je souhaite remettre à mes clients (option fidélisation)

Dans le tableau de bord, la rubrique Gestion des récompenses permet de changer le pourcentage de cashback sur le Montant total de la facture (avec les taxes, avant le pourboire) que vous souhaitez remettre à vos clients qui payent avec Trendigo. Tous les pourcentages peuvent être changés, sauf celui du jour.

Accéder à mes rapports (quotidiens ou mensuels)

L’onglet gestion des rapports permet un accès à toutes les transactions réalisées. Il est possible de faire une annulation (VOID) jusqu’à une heure après la transaction. Un remboursement est possible jusqu’à 30 jours.

Comment créer un compte pour mes employés ?

Dans le tableau de bord, la rubrique collaborateur (employé) vous permet de créer un compte Collaborateur (employé). Il pourra ainsi utiliser l’Application mobile TrendigoM pour suivre les paiements et créer un code QR dynamique incluant le montant de la facture. Le client devra simplement scanner le code QR produit par l’Application pour payer. Ces comptes de collaborateurs (employés) peuvent être supprimés très simplement depuis le tableau de bord en cliquant sur la poubelle.

Quand l’argent reçu de mes clients est-il versé sur mon compte ?

Les paiements sont versés sur votre compte 1 à 2 jours ouvrés après que la transaction soit passée avec succès.

Paiement

Comment ça marche ?

Trendigo est un moyen de paiement et de fidélisation dématérialisé. Le terminal de paiement est dans le mobile de vos clients. Un client qui souhaite vous payer scannera votre code QR unique. Dans son application, il choisit le montant du pourboire et lance la transaction. Vous savez immédiatement par une notification, un courriel de confirmation qu’elle a été acceptée. Elle s’ajoute également dans la liste de vos transactions sur la page principale du tableau de bord et dans votre application marchand (si vous choisissez de l’utiliser). Ce client reçoit un pourcentage de sa facture en cashback qu’il pourra réutiliser pour un futur paiement. C’est vous qui déterminez le montant du cashback. L’argent de vos transactions moins les frais de transaction est versé directement sur votre compte bancaire. Vous gardez le montant de cash-back dans votre caisse, mais votre client pourra l’utiliser au moment qu’il aura choisi.

Récupérer mon code QR unique

Le code QR unique est placé en haut à gauche de la page d'accueil du tableau de bord. Il peut être imprimé ou téléchargé pour être placé au meilleur endroit possible pour vous. Ces actions ne sont possibles qu’après validation de votre compte. Au moment de votre inscription, nous vérifions la conformité de votre établissement. Une fois ce processus réalisé (de 1 heure à 24 heures), vous avez accès au code QR.

Créer un Code QR de paiement

En plus du code QR unique, vous pouvez créer des codes QR dynamiques à partir de l’application Marchand (disponible également sur les boutiques en ligne iOS et Android). Connectez-vous avec vos identifiants, cliquez sur le bouton « créer un code QR », entrez le montant total de la facture due (avec les taxes). Votre client scannera ce code pour payer.

Quelles sont les cartes acceptées ?

Trendigo accepte la plupart des cartes de crédit ou de débit. Mastercard, Visa, American Express. D’autres cartes seront acceptées dans le futur.

Où est transféré l’argent donné en cash-back à mes clients ?

L’argent donné à vos clients est inclus dans leur porte-monnaie électronique. Ils peuvent l’utiliser dès leur prochain paiement. Le montant reste dans votre caisse. Vous pouvez suivre le montant donné à vos clients et utilisé dans le dashboard du portail marchand.

Où mes clients peuvent-ils utiliser l’argent reçu en cash-back ?

Selon l’option choisie, vos clients peuvent l’utiliser soit dans votre commerce uniquement, soit au sein d’un réseau de commerçants que vous avez rejoint ou créé.

Réseau de marchands

Quelle différence entre la communauté Trendigo et les réseaux privés de commerçants Trendigo ?

La communauté Trendigo est composée de tous les commerçants et clients qui ont choisi d’accepter le paiement trendigo. Il est basé sur la découverte des commerces de proximité et l’aide au commerce local. Un commerçant ou un réseau de commerçants (centre commercial, association de commerces, franchise) peut choisir le programme de fidélisation trendigo. Le cash-back remis à leurs clients pourra être utilisé uniquement chez eux. Les clients pourront toujours payer chez les autres commerces Trendigo, mais ne pourront pas utiliser ces récompenses ailleurs.

Comment créer mon propre réseau de commerces ?

Pour créer un réseau de commerces, commencez l’inscription. Après vérification de votre courriel, il vous sera demandé de choisir la communauté que vous souhaitez rejoindre. Choisissez « Créer une communauté », indiquez son nom et finissez votre inscription. Vous pourrez ensuite donner ce nom aux commerces que vous souhaitez ajouter pur qu’ils s’enregistrent également.

Comment rejoindre un réseau de commerçants existants ?

Pour rejoindre un réseau de commerçants existants, le créateur et gestionnaire du programme vous communiquera le nom de son programme. Commencez le processus d’inscription, choisissez « rejoindre une communauté » et sélectionnez-la dans la liste puis complétez l’inscription.
